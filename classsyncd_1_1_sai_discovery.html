void SaiDiscovery::discover ( _In_ sai_object_id_t rid,
_Inout_ std::set< sai_object_id_t > & processed )
private

Discover objects on the switch.

Method will query recursively all OID attributes (oid and list) on the given object.

This method should be called only once inside constructor right after switch has been created to obtain actual ASIC view.

Parameters
ridObject to discover other objects.
processedSet of already processed objects. This set will be updated every time new object ID is discovered.
