Detailed Description

Skip record attributes container.

This container will hold attributes that can be skipped when recording GET api, for example statistics attributes and *_AVAILABLE_*.

Since orch agent can query those frequently it can produce large recording files without meaningful information.

Any attribute can be added to this container, but only non oid attributes will be accepted.
