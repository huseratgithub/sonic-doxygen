Detailed Description

Class represents single attribute.

Some attributes are lists, and have allocated memory, this class will help to handle memory management and also will keep metadata for attribute.

Get OID list from attribute.

Based on serialization type attribute may be oid attribute, oid list attribute or non oid attribute. This method will extract all those oids from this attribute and return as vector. This is handy when we need processing oids per attribute.
