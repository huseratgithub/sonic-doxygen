Clear OID values.

For every OID attribute on list will set their oids to SAI_NULL_OBJECT_ID. This is handy, since when performing GET operation, user may not clear allocated LIST, and GET also serializes full list contents, so exact buffer will be deserialized on syncd side.

This operation is more for a convenience use, but we should create serialize and deserialize special for GET operation.
