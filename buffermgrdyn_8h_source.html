36// State of the profile.
37// We maintain state for dynamic profiles only
38// INITIALIZING state indicates a profile is under initializing
39// NORMAL state indicates that profile is working normally
40// A profile's state will be STALE When it is no longer referenced
41// It will be removed after having been in this state for 1 minute
42// This deferred mechanism yields two benefits
43// 1. Avoid frequently removing/re-adding a profile
44// 2. Avoid removing profile failure if orchagent handles BUFFER_PG
45// and BUFFER_PROFILE tables in an order which differs from which
46// buffermgrd calls

48 PROFILE_INITIALIZING,
